Unity-DAY1 3D基礎

Author: Niniiii Date: July 4, 2022  23:38:58 Category: Unity學習

Unity是什麼??

“是一種跨平台的2D和3D遊戲引擎” –維基百科

那遊戲引擎又是什麼呢 ??

“遊戲引擎就是一個程式庫，負責幫助團隊處理所有遊戲開發相關的事宜。”

【專欄】淺談製作遊戲最重要的開發工具「遊戲引擎」《Qubot》 - 巴哈姆特 (gamer.com.tw)

(這篇文章大部分內容關於遊戲引擎的歷史演進)

其實就是 :

  • 具有可視化界面創建遊戲
  • 程式庫提供，可以方便遊戲建構許多功能(包含 : 渲染、地形、光影等等)

下載Unity

  1. Unity官網下載Unity Hub(管理版本跟模組)

  2. 安裝完畢後開啟，應該會直接詢問你要下載哪一個版本的Unity

    我是選擇最新的並且有(LTS : long-term support)標籤的版本

  3. 接下來會詢問你要下載那些模組?

    • Android Build Support

    • WebGL : 可以讓你的遊戲能夠在網站上運作。

Unity介面

開啟新的project

  • 開啟unity hub > project > 點選又上的按鈕new project

  • 選擇你要的unity版本，3D templates，輸入project name和選擇資料夾位置，按下create project

介面區塊

unity介面主要會分成這幾個區塊 : Main Menu、Toolbar、Hierarchy、Project、Inspector、Scence

  • Toolbar : 可以放上你最常使用到的功能

  • Hierarchy : 顯示遊戲的game object

  • Inspector : 特定game object的內容，包含一些可以調整的參數還可以新增一些元素

  • Sence : 場景畫面

  • Project : 專案資料夾內容

基礎操作

新增一個物件

在Hierarchy區塊裡右鍵尋找

scene view”物件”操作

scene view”視角”操作

  • 環顧周圍 : 右鍵
  • 拉近拉遠 : 滾輪、alt + 右鍵
  • 尋找到特定物件 : 雙擊Hierarchy區塊中你要找的物件
  • 環視物件 : alt + 左鍵

建立game object組合

  1. 新增一個Empty(右鍵 > create empty)
  2. 可以設定empty位置以及其他參數
  3. 將子物件全部都放到empty底下

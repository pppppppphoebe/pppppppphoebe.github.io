hexo 介紹

hexo是一個靜態的網誌框架

(何謂靜態 : 是指不用後台，只用html, css, javascript即可生成的網站)

特點:

  • 超快的檔案產生速度，在幾秒內即可完成網誌產檔

  • 近300個外掛插件

  • 有200多個免費主題

  • Markdown支援

將部落格佈署在github

  1. 先在自己的github上，創建一個新的Repository，名子一定要是 : 用戶名.github.io

  2. 配置ssh key

    完成後，查看ssh

    首次使用要添加主機到本機的ssh可信列表

    要顯示”Hi xxx” You’ve successfully….”才是成功哦!

  3. 在你的github添加剛剛生成的ssh key

    找到settings>SSH and GPG keys 並點擊由上方的New SSH key
